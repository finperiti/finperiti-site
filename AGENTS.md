## Critical Rules

### 🚫 DO NOT:
- Invent marketing copy or messaging
- Guess at brand capitalization (use seo.md)
- Skip validation before committing
- Commit without a ticket reference

### ✅ DO:
- Start with [.dev/index.md](.dev/index.md)
- Consult [seo.md](.dev/docs/prds/seo.md) for all copy
- Follow the documentation hierarchy
- Test changes before committing
- Include ticket references (WRK-21, MRK-20, etc.)
